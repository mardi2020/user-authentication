### 👩🏻‍🔧 개요
사용자 인증 시스템은 회원가입부터 내정보 관리, 관리자의 사용자 관리를 지원하는 웹 기반 시스템입니다. 또한, 사용자들을 그룹으로 묶어 관리할 수 있습니다.

### 🛠 기술
- language: Java11, JavaScript
- framework: Spring boot 2.7.6, React
- database: MySQL 8.0, Redis
- devOps: Docker, Apache Kafka
